# Hula
Hula is a real-time chat application build using Socket.io

## Tech Stack

**Client:** ReactJS, HTML, CSS

**Server:** NodeJS, ExpressJS

**Database:** MongoDB




## Run Locally

Clone the project

```bash
  git clone  https://github.com/unknownuser2329/Hula
```

Go to the project directory

```bash
  cd Hula
```

Install dependencies

```bash
  npm install
```
```bash
cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```

Start the Client

```bash
cd frontend/
  npm start
```
